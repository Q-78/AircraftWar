# Aircraft War 飞机大战

本项目为哈尔滨工业大学（深圳）《软件构造》课程实验项目，基于 Java Swing 实现一款飞机大战小游戏。项目从基础模板逐步迭代，综合应用了面向对象设计、UML 建模、设计模式、文件持久化、单元测试、Swing 图形界面、多线程音效控制、观察者模式和模板方法模式等内容。

在课程基本要求的基础上，本项目还增加了多项交互优化与创新功能，包括暂停系统、主动大招、无敌冲刺、Combo 连击系统、Boss 登场警告动画、Boss 血量条、技能冷却条、道具生效提示、排行榜查询等，使游戏体验更加完整。

---

## 一、项目简介

游戏中，玩家控制英雄机躲避敌机和敌机子弹，并通过发射子弹击毁敌机获得分数。不同敌机具有不同的移动方式、生命值、攻击方式和得分奖励。敌机被击毁后可能掉落道具，玩家拾取道具后触发对应效果。

游戏支持三种难度模式：

- 简单模式
- 普通模式
- 困难模式

不同难度拥有不同的敌机生成规则、Boss 生成规则、游戏递进逻辑和排行榜数据。

## 三、项目结构

```text
src/
└── edu/
    └── hitsz/
        ├── aircraft/       # 飞机类，包括英雄机和各类敌机
        ├── application/    # 游戏主流程、界面、音效、排行榜等
        ├── basic/          # 飞行物抽象父类
        ├── bullet/         # 子弹类
        ├── dao/            # 排行榜 DAO 数据访问层
        ├── factory/        # 敌机工厂、道具工厂
        ├── observer/       # 观察者模式相关接口
        ├── prop/           # 道具类
        ├── strategy/       # 射击策略类
        ├── test/           # 单元测试
        ├── images/         # 图片资源
        ├── videos/         # 音效资源
        └── uml/            # PlantUML 类图文件
